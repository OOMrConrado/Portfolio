# Portfolio

Personal portfolio website built with Next.js, showcasing projects, experience, and technical skills.

## Tech Stack

- Next.js 15
- React 19
- Tailwind CSS 4
- DaisyUI
- Devicon

## Features

- Responsive design
- Dark/Light mode support
- Animated tech stack marquee
- SEO optimized
- Contact form integration
- Project showcase

## Setup

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Configuration

Edit `src/data/data.js` to customize your portfolio content.
